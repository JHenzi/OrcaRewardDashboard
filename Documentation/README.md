# Documentation Directory

This directory contains organized documentation for the Orca Redemption Tracker project.

## Directory Structure

### `/archived`
Completed fixes, resolved issues, and historical status documents:
- `AUXILIARY_HEADS_FIX.md` - Fix for NaN weights in auxiliary prediction heads
- `AUXILIARY_LOSSES_DISABLED.md` - Documentation on temporarily disabling auxiliary losses
- `TRAINING_NAN_FIX.md` - Enhanced NaN protection during training

Reference documentation, guides, strategies, and historical information:
- `APP_FLOW_AUDIT.md` - Application flow audit and architecture
- `APP_FLOW_DIAGRAMS.md` - Visual flow diagrams

- `CHANGELOG_SOL_TRACKER.md` - Changelog for SOL Tracker features

- `CURRENT_SYSTEM_STATUS.md` - Current system status snapshot
- `DEPRECATED.md` - Documentation of deprecated features
- `MLOPS_STRATEGY.md` - MLOps strategy for RL agent
- `NewAgent.md` - Original RL agent design specification
- `PREDICTIONS_EXPLAINED.md` - Explanation of RL agent predictions
- `RETRAINING_STRATEGY.md` - Automated retraining strategies
- `TRAINING_FAQ.md` - Frequently asked questions about training
